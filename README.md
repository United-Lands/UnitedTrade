# UnitedTrade
UnitedTrade is a plugin for geopolitical Minecraft servers, first created for UnitedLands (play.unitedlands.net). The core feature is the creation and management of trade points where the players will be offered randomly generated trade contracts to fulfill for money, with time limitations and optional contractual penalties.

Visit the  [Wiki](https://github.com/United-Lands/UnitedTrade/wiki) for details.
